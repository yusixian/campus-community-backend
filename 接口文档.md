# 接口设计
### 用户模块
##### 注册接口
```bash
POST/user/register
```
- 请求参数
```
user_name,password
```
- 响应
**成功**
```JSON
{
  "code":0,
  "message":"用户注册成功",
  "result":{
    "id":2,
    "user_name":"user"
  }
}
```
##### 登录接口
```bash
POST/user/login
```
- 请求参数
```
user_name,password
```
- 响应
**成功**
```JSON
{
  "code":0,
  "message":"用户登录成功",
  "result":{
    "token":"...省略"
  }
}
```
